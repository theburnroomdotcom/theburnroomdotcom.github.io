# Videos Directory

Place your videos here. Recommended structure:

```
videos/
├── hero-bg.mp4            # Hero section background video
└── kitchen-reel.mp4       # Kitchen/cooking process video
```

## Video Guidelines

### Hero Background Video

- **Format**: MP4 (H.264 codec)
- **Duration**: 15-30 seconds, seamlessly looping
- **Resolution**: 1920x1080px minimum
- **Content suggestions**:
  - Slow-motion cooking flames
  - Plating process close-up
  - Ambient dining atmosphere
  - Ingredient preparation

### General Tips

1. Keep videos short (under 30 seconds for backgrounds)
2. Compress videos for web (aim for under 10MB)
3. Use tools like HandBrake to optimize file size
4. Ensure videos loop smoothly without a visible jump
5. Audio will be muted for background videos

## Alternative: YouTube/Vimeo Embeds

For longer content (event recaps, chef interviews), consider embedding from YouTube or Vimeo instead of self-hosting. Update the HTML with the embed code.
