# Assets Directory

Place your images here. Recommended structure:

```
images/
├── hero-fallback.jpg       # Fallback image for hero section (1920x1080px)
├── chef-portrait.jpg       # Photo of Keshav Menon (800x1000px)
└── gallery/
    ├── featured.jpg        # Large featured gallery image (800x1000px)
    ├── dish-1.jpg          # Individual dish photos (400x300px each)
    ├── dish-2.jpg
    ├── ambiance-1.jpg      # Atmosphere/setting shots
    ├── kitchen-1.jpg       # Behind-the-scenes photos
    └── ...
```

## Image Guidelines

- **Hero fallback**: Wide landscape shot of the dining experience or food close-up. Recommended: 1920x1080px
- **Chef portrait**: High-quality portrait of Keshav Menon. Recommended: 800x1000px (portrait orientation)
- **Gallery images**: Mix of dish close-ups, table settings, and atmosphere shots. Various sizes depending on layout position.

## Tips for Best Results

1. Use high-resolution images (at least 2x the display size for retina screens)
2. Optimize images before uploading (use tools like TinyPNG or ImageOptim)
3. Use JPEG for photos, PNG for graphics with transparency
4. Keep file sizes under 500KB for web performance
